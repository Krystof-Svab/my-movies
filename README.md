# Knihovna filmu

Webova aplikace v Next.js App Routeru pro spravu filmu ulozenych v Supabase.

## Funkce

- seznam filmu na `/movies`
- filtrovani filmu podle zanru
- detail filmu na `/movies/[id]`
- pridani filmu na `/movies/new`
- uprava filmu na `/movies/[id]/edit`
- smazani filmu z detailu
- formulare s `react-hook-form` a `Zod`

## Spusteni projektu

Nainstalujte zavislosti:

```bash
npm install
```

Vytvorte soubor `.env.local` a doplnte Supabase udaje:

```env
NEXT_PUBLIC_SUPABASE_URL=vas_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=vas_supabase_anon_key
```

Spustte vyvojovy server:

```bash
npm run dev
```

Aplikace pobezi na:

```txt
http://localhost:3000
```

## Supabase tabulka

V Supabase vytvorte tabulku `movies` se sloupci:

```sql
create table movies (
  id bigint generated by default as identity primary key,
  title text not null,
  director text not null,
  year integer not null,
  genre text not null,
  rating numeric not null,
  duration time not null
);
```

Pokud mate zapnute Row Level Security, pridejte policy pro cteni, vkladani, upravy a mazani pro `anon` uzivatele, nebo RLS pro skolni testovani vypnete.

## Build

Pro kontrolu produkcniho buildu:

```bash
npm run build
```
